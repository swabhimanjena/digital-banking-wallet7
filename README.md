# 🏦 Digital Banking Wallet

A secure Digital Banking Wallet application built using **Python**, **FastAPI**, **SQLAlchemy**, **JWT Authentication**, and **SQLite**.

---

## 🚀 Features

- 🔐 User Registration
- 🔑 User Login
- 🛡 JWT Authentication
- 💰 Wallet Dashboard
- ➕ Deposit Money
- ➖ Withdraw Money
- 📜 Transaction History
- 🎨 Responsive HTML Templates

---

## 🛠 Tech Stack

- Python
- FastAPI
- SQLAlchemy
- SQLite
- JWT
- Jinja2
- HTML
- CSS
