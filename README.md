# Digital Banking Wallet

A secure Digital Banking Wallet application built using Python, FastAPI, SQLAlchemy, JWT Authentication and SQLite.

## Features

- User Registration
- User Login
- JWT Authentication
- Wallet Dashboard
- Deposit Money
- Withdraw Money
- Transaction History

## Tech Stack

- Python
- FastAPI
- SQLAlchemy
- SQLite
- JWT
- Jinja2
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
python main.py
